A web login manager that uses a JS file to open and automatically login into your account using your discord Token

DISCLAIMER!!!
Use responsively, I am not responsible for any bans or blacklists that you get while using this

How to use:
1. Install the tapermonkey script by copying the contents of JS helper script into a new tapermonkey/whatever script editor you use
     or
   Drag and drop the JS script into Your script editor

2. Add discord tokens and name on website
3. Click open and it will automatically login with the pasted discord tokens account
